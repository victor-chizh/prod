# Victor Chizh — Portfolio

Одностраничный сайт-портфолио (DOP / Filmmaker / Edit). Тёмная кинематографичная тема, двуязычный интерфейс (RU/EN), автопрокручивающиеся карусели работ и вертикальных видео.

## Структура

```
index.html              — главная страница (весь сайт в одном файле: HTML/CSS/JS)
assets/
  videos/                — фоновые и превью-видео
  images/                — логотипы брендов и превью-кадры проектов
```

## Как открыть локально

Просто откройте `index.html` в браузере — никакой сборки или сервера не требуется.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите содержимое этой папки (сохраняя структуру `assets/`).
2. В настройках репозитория откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main` и папку `/ (root)`.
4. Сохраните — через пару минут сайт будет доступен по адресу вида `https://<username>.github.io/<repo-name>/`.

## Технологии

Чистый HTML/CSS/JS без сборки и зависимостей. Шрифты подключены как системные fallback-стеки (Oswald, IBM Plex Mono, Work Sans) — при желании можно подключить их через Google Fonts в `<head>`.
